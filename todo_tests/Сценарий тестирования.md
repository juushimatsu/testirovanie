# Проверка функционала добавления задачи

## Цель:
Убедиться что пользователь может добавить задачу и она сохраняется корректно

## Предпосылки:
- Файл с таксками отстутствует/пуст/содержит какие-то данные
- Пользователь находится в главном меню

## Шаги тестирования

1. В главном меню выьрать пункт 2 (Добавить задачу)
- Ожидаемый результат:
"Введите описание задачи"

2. Ввести описание задачи, пример "Тест".
- Ожидаемый результат:
"Задача Тест - добавлена", возврат в главное меню

3. Выбрать пункт 1 (Все задачи)
- Ожидаемый результат: 
В списке задача с id 1, описанием "Тест", статус "-"(не выполнена)

## Фактический результат: 
Тест пройден успешно, все шаги тестирования выдали ожидаемый результат.
